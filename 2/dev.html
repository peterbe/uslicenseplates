About this app

This app was built by, yours truely, Peter Bengtsson (www.peterbe.com) because having recently moved to the USA I found myself searching for new out of state license plates I've never seen before. But I had to log that somewhere. And what better way than to write a fun little mobile app to do that.

All code for this project is Open Source and available at github.com/peterbe/uslicenseplates.

To be honest, this app is more of an experiment than anything else. The intention is actually to re-implement this app idea in multiple different ways with different types of technology so as to learn how to use and and which of these tools is best.

The Rules

Unlike many of games this one has no enforecable rules. Since you don't have to prove how or where you spotted a plate it's up to you to be truthful to yourself.

My rule is that you have to clearly be able to see the license plate with your own eyes.
